Introdução

Descrição

No âmbito da Unidade Curricular de Laboratório de Sistemas do 1º Semestre do 3º ano da Licenciatura, foi desenvolvido um sistema de monitorização do clima.

No contexto da "Smart Home" (Casa Inteligente) e da domótica, a monitorização do conforto térmico e das condições exteriores é fundamental para um bem-estar pessoal. Este projeto propõe o desenvolvimento de uma Estação Meteorológica Residencial.

Adicionalmente, o sistema deteta chuva para automatizar a recolha de um estendal.

Arquitetura

A arquitetura do sistema baseia-se numa topologia centralizada em estrela, onde o microcontrolador ATmega328P atua como mestre, gerindo a aquisição de dados e o controlo de atuadores através de barramentos digitais dedicados.

  • Barramento I2C: Os dois sensores (AHT20, BMP280) e o ecrã LCD partilham as mesmas duas linhas de comunicação (SDA e SCL). O microcontrolador distingue cada dispositivo através do seu endereço digital.
  • Controlo PWM (Pulse Width Modulation): O servomotor é controlado pelo Timer1 (Hardware) do ATmega328P. Isto garante movimentos precisos sem ocupar ciclos de processamento da CPU.
  • I/O: A leitura do sensor de chuva e o acionamento dos LEDs são realizados através de manipulação direta de bits nos registos PINC (Entrada) e PORTD/PORTB (Saída).

Hardware

Neste capítulo são apresentados os componentes selecionados para o projeto, detalhando a sua função no sistema.

Microcontrolador ATmega328P

O "cérebro" do sistema é o ATmega328P, um microcontrolador de 8 bits da arquitetura AVR. A sua escolha deve-se ao equilíbrio entre performance e baixo consumo energético. Operando a 8MHz (oscilador interno), ele gere todos os protocolos de comunicação (I2C, UART) e temporizadores necessários. A programação "Bare Metal" permite utilizar a totalidade dos 32KB de memória Flash de forma otimizada.

Sensores Ambientais (Barramento I2C)

Sensor AHT20 (Temperatura e Humidade):

Escolhido pela sua fiabilidade e calibração de fábrica. Comunica via I2C, eliminando a necessidade de protocolos "single-wire" complexos. Fornece os dados base para o cálculo do Índice de Calor ("Real Feel").

Sensor BMP280 (Pressão Barométrica):

Desenvolvido pela Bosch, este sensor piezo-resistivo mede a pressão atmosférica com elevada precisão. Utiliza o endereço I2C 0x77 e permite monitorizar a pressão em hPa (Hectopascais), útil para inferir tendências meteorológicas.

Interface e Atuadores

Ecrã LCD 20x4 (c/ Adaptador I2C):

Para a visualização dos dados, optou-se por um LCD de 4 linhas. A inclusão de um adaptador I2C (baseado no chip PCF8574) é crucial, pois reduz o número de pinos necessários de 6 (modo paralelo) para apenas 2 (SDA/SCL).

Servomotor SG90:

O servo simula o mecanismo de proteção (teto retrátil), controlado por PWM a 50Hz.

Esquema Elétrico

O circuito é alimentado por uma bateria de 9V, regulada para 5V através de um regulador linear (ex: LM7805) para alimentar o ATmega328P e os sensores.
